# 算点Janten 隐私政策

生效日期：2026-07-07

「算点Janten」（下称"本应用"）是一款日本麻将（立直麻将）计分辅助工具。我们深知隐私的重要性，本政策说明本应用如何处理你的信息。

## 一句话总结

**本应用不收集、不存储、不上传任何个人信息。**

## 详细说明

- **相机与照片**：拍摄或选取的牌面照片仅在你的设备本地通过端侧模型进行识别，照片与识别结果均不会上传到任何服务器。
- **对局数据**：记分簿中的对局记录仅保存在你的设备本地，删除应用即彻底清除。
- **网络**：本应用核心功能完全离线运行，不包含任何数据上报、广告或第三方分析 SDK。
- **权限**：相机权限仅用于拍摄牌面识别，可随时在系统设置中关闭，不影响手动录入功能。
